The Plugin Wel!Amazon Adds is an easy way to include Amazon links into your WordPress Blog.

== Description ==

The Plugin Wel!Amazon Adds is an easy way to include Amazon links into your WordPress Blog. It enables the integration of Image Links and Extended Links directly from the Amazon database. These links can be placed into the text or on the sidebar, so that each page gets a suitable sidebar for the topic of the content. For more flexibility, the Amazon links can be displayed as a randomized link or crossfaded several links.

== Installation ==

This section describes how to install the plugin and get it working.

Make sure your Wordpress version is equal or greater than 2.7 and your hosting provider is using PHP5.

Upload the `welamazonadds` folder along with all its files to the `/wp-content/plugins/` folder

Activate the plugin from the **Plugins** menu in WordPress Dashboard

Configure the plugin: 
- Go to the **Settings** menu in WordPress Dashboard
- Select **Wel!Amazon Affiliate**
- Setup your Partner-ID/Tracking-ID and your Location. The  Partner or Tracking-ID is the Username for Amazon Associate or Partner Net (Not your Amazon Username). You can find it on left top side at the Amazon Associate Webpage.
- For using images, you have to set your Set your Access Key ID and Secret Access Key. You can create a free account on http://aws.amazon.com/.
- After it you can setup some layout options.

To activate the Widget on the sidebar goto **Appearance** ans select **Widgets**. Drag and drop the `Wel!Amazon Affiliate` widget to your sidebar. To configure the widget activate the drop down and setup the options.
